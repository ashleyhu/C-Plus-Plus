Step 1. Compile and run <matrices.cpp> <matrixmath.cpp> and <matrixmath.h>.
Step 2. Program reads in the dimensions and values of 2 matrices in input.txt.
Step 3. Program checks that dimensions read in matches the dimensions of the matrices.
Step 4. Program echos input data to screen.
Step 5. Program checks that the matrices are multiplicable.
Step 6. Program performs multiplication of 2 matrices.
Step 7. Program outputs result of matrix multiplication to screen and to output.txt.
Step 8. Press any key to dismiss the window.
