Instructions to run the program:

Step 1. Compile and run <Pi.cpp>.
Step 2. The pi approximation calculation stops when the nth decimal place has settled to a consistent number after three iterations. Please enter a value for n between 1 to 6 inclusive.
Step 3. Program displays term number and value of pi approximation until 3 consecutive iterations of the same nth decimal place has been outputted.
Step 4. Press any key to dismiss the window.
